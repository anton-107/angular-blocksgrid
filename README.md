#angular-blocksgrid

AngularJS directive for creating grids and allowing users to drag-and-drop elements of this grid

##Installation

###Bower

`bower install angular-blocksgrid`

##Development

###Requirements

- node and npm
- bower `npm install -g bower`
- grunt `npm install -g grunt-cli`

###Setup

- `npm install`
- `bower install`

###Run

`grunt dev`

or for just running tests on file changes:

`grunt ci`

###Tests

`grunt mocha`
